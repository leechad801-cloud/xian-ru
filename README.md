# Xian-ru

一个面向手机的黑色极简个人音乐 + 阅读 + 笔记网站，基于 React、TypeScript、Tailwind CSS 和 epubjs 构建。

## 功能

- **Kindle 风格阅读器**：支持通过 Google Drive 公开链接读取 EPUB / TXT，支持字体大小、行高、字体、浅深色主题与进度保存。
- **全局音乐播放器**：76 首歌曲可在任意页面持续播放，支持顺序 / 循环 / 随机、音量调节与进度控制。
- **本地笔记**：笔记内容使用 LocalStorage 自动持久化。

## 本地开发

```bash
npm install
npm run dev
```

## 校验

```bash
npm run lint
npm run build
```
